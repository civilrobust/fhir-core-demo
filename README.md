# Clinical FHIR Integration Demo (React + Vite)

A polished, clinical-style demo that:
- searches FHIR (Fast Healthcare Interoperability Resources) Patient resources
- shows Patient details
- pulls Observation resources (vitals / labs)
- generates an AI (Artificial Intelligence) summary + alerts + next-actions (demo)

## Run
npm install
npm run dev

## Deploy (Cloudflare Pages)
Build command: npm run build  
Output folder: dist
